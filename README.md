### Steps for Project Execution :

##### Install Required Packages:
```
1. Beep - go get -u github.com/faiface/beep
2. TCell - go get -u github.com/gdamore/tcell
```
##### To Run:
```
1. Open terminal
2. Enter:
    go run music-CLI.go ./song.mp3
    This will play the audio on the CLI !
```
